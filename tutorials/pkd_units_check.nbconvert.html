MT Data are typically reported with magnetic fields in nanoTesla (nT) and electric fields in milliVolts-per-kilometer (mV/km).

In this notebook, some field data from a relatively quiet site (Parkfield, California) is accessed via MTH5, and information about the units is emphasized.

Starting with an MTH5 with data in units of Digital Counts, the embedded filter information is displayed for all channels including the input and output units of each filter, and their coefficients.

This is intended as a reference that can be searched for where specific pieces of metadata information are stored and how they look when mt_metadata renders them.

Then the data are calibrated using MTH5âs method and the data are plotted in digital counts and the MT units.
